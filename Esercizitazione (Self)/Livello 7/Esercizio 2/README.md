# Esercizio 2: Gestione Dipendenze e JDBC / Exercise 2: Dependency Management and JDBC

## 🇮🇹 Italiano

### Traccia
Inserisci nel `pom.xml` le dipendenze per connetterti a un database relazionale usando JDBC (es. driver MySQL o HSQLDB) e testa la connessione.

### Esempio di Output
```text
Connecting to database...
Connection successful!
Executing test query: SELECT 1
Result: 1
Closing connection.
```

---

## 🇬🇧 English

### Syllabus
Insert the dependencies in the `pom.xml` to connect to a relational database using JDBC (e.g., MySQL or HSQLDB driver) and test the connection.

### Sample Output
```text
Connecting to database...
Connection successful!
Executing test query: SELECT 1
Result: 1
Closing connection.
```
