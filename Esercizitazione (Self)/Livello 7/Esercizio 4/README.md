# Esercizio 4: Architettura REST (Spring MVC) / Exercise 4: REST Architecture (Spring MVC)

## 🇮🇹 Italiano

### Traccia
Aggiungi le dipendenze di Spring MVC al progetto Maven. Definisci un controller base che risponda a una richiesta HTTP (simulando un'architettura web).

### Esempio di Output
**Request:** `GET /api/hello`

**Response:**
```json
{
  "message": "Hello World",
  "status": "success",
  "timestamp": "2023-10-27T10:00:00Z"
}
```

### Syllabus

### Sample Output

**Response:**
```json
{

}
```
