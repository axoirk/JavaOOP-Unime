# Esercizio 6

---

## 🇮🇹 Italiano

### Descrizione
Questo esercizio lavora su input da tastiera e gestione di array numerici.
Il programma acquisisce da utente 7 temperature giornaliere, le memorizza in un array, ne calcola la somma e la media settimanale.

### Esempio di Output
```
Inserisci temperatura del giorno 1: 18.5
Inserisci temperatura del giorno 2: 20.0
...
La media della temperatura settimanale è: 19.27
```

---

## 🇬🇧 English

### Description
This exercise focuses on keyboard input and numeric array processing.
The program reads 7 daily temperatures from user input, stores them in an array, and calculates the total sum and weekly average.

### Output Example
```
Inserisci temperatura del giorno 1: 18.5
Inserisci temperatura del giorno 2: 20.0
...
La media della temperatura settimanale è: 19.27
```
