# Esercizio 3

---

## 🇮🇹 Italiano

### Descrizione
Usa un costrutto `switch` basato su costanti o enumerazioni per stampare un messaggio diverso in base al voto (A, B, C, D, F) di uno studente.

#### Requisiti:
- 'A': "Eccellente"
- 'B': "Molto Buono"
- 'C': "Sufficiente"
- 'D': "Scarso"
- 'F': "Bocciato"
- Gestire il caso `default` per voti non validi.

### Esempio di Output
```
Inserisci il voto dello studente (A-F): B
Valutazione: Molto Buono
```

---

## 🇬🇧 English

### Description
Use a `switch` construct based on constants or enumerations to print a different message based on a student's grade (A, B, C, D, F).

#### Requirements:
- 'A': "Excellent"
- 'B': "Very Good"
- 'C': "Sufficient"
- 'D': "Poor"
- 'F': "Failed"
- Handle the `default` case for invalid grades.

### Output Example
```
Inserisci il voto dello studente (A-F): B
Valutazione: Molto Buono
```
