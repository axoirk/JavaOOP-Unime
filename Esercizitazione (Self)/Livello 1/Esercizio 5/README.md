# Esercizio 5

---

## 🇮🇹 Italiano

### Descrizione
Crea due classi in due file separati. Inserisci la prima in un package specifico usando la direttiva `package` all'inizio del file, e importala nella seconda classe usando `import`.

#### Requisiti:
- Definire una classe di utilità in un package (es. `com.math.Utils`).
- Creare una classe `Main` in un altro package che utilizzi la classe di utilità.

### Esempio di Output
```
Risultato dell'operazione inter-package: 25
```

## 🇬🇧 English

### Description
Create two classes in two separate files. Place the first one in a specific package using the `package` directive at the beginning of the file, and import it into the second class using `import`.

#### Requirements:
- Define a utility class in a package (e.g., `com.math.Utils`).
- Create a `Main` class in another package that uses the utility class.

### Output Example
```
Risultato dell'operazione inter-package: 25
```
