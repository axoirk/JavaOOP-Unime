# Esercizio 1 – Ereditarietà / Inheritance

---

## 🇮🇹 Italiano

### Descrizione
Questo esercizio esplora i concetti fondamentali dell'**ereditarietà** e della relazione "is-a" tra classi.

Il programma:
- Definisce una superclasse `Person` con attributi base come il nome.
- Estende `Person` con una sottoclasse `Student`, aggiungendo attributi specifici come il numero di matricola.
- Dimostra l'uso di `super()` per richiamare il costruttore della superclasse.
- Illustra il **polimorfismo**: un riferimento di tipo `Person` può puntare a un oggetto di tipo `Student`.

### Obiettivo didattico
- Comprendere la gerarchia delle classi in Java.
- Utilizzo della keyword `extends`.
- Override dei metodi per specializzare il comportamento delle sottoclassi.

### Esempio di Output
```text
Name: Giulia
Name: Marco
Student Number: 12345
```

---

## 🇬🇧 English

### Description
This exercise explores the fundamental concepts of **inheritance** and the "is-a" relationship between classes.

The program:
- Defines a `Person` superclass with basic attributes like name.
- Extends `Person` with a `Student` subclass, adding specific attributes like a student ID number.
- Demonstrates the use of `super()` to call the superclass constructor.
- Illustrates **polymorphism**: a `Person` reference can point to a `Student` object.

### Learning Objective
- Understand class hierarchy in Java.
- Use of the `extends` keyword.
- Method overriding to specialize subclass behavior.

### Output Example
```text
Name: Giulia
Name: Marco
Student Number: 12345
```
