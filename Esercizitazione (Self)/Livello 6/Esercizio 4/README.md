# Esercizio 4: Java e XML (DOM) / Exercise 4: Java and XML (DOM)

## 🇮🇹 Italiano

### Descrizione
Crea un file XML con una lista di studenti. Usa `DocumentBuilder` per caricarne l'albero in memoria (Object Model) e itera sui nodi per stamparli.

### Esempio di Output
```text
Lista Studenti (DOM):
- Nome: Mario Rossi, Matricola: 12345
- Nome: Giulia Bianchi, Matricola: 67890
```

---

## 🇬🇧 English

### Description
Create an XML file with a list of students. Use `DocumentBuilder` to load the tree into memory (Object Model) and iterate over the nodes to print them.

### Sample Output
```text
Student List (DOM):
- Name: Mario Rossi, ID: 12345
- Name: Giulia Bianchi, ID: 67890
```
