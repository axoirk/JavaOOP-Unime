# Esercizio 2: Client/Server (Socket TCP) / Exercise 2: Client/Server (Socket TCP)

## 🇮🇹 Italiano

### Descrizione
Crea un programma Server che resta in ascolto su una porta specifica. Crea un programma Client che si connette e invia un messaggio usando la comunicazione orientata alla connessione.

### Esempio di Output
```text
[Server] In attesa di connessione...
[Server] Client connesso: /127.0.0.1
[Server] Messaggio ricevuto: Ciao dal Client TCP!
```

---

## 🇬🇧 English

### Description
Create a Server program that listens on a specific port. Create a Client program that connects and sends a message using connection-oriented communication.

### Sample Output
```text
[Server] Waiting for connection...
[Server] Client connected: /127.0.0.1
[Server] Message received: Hello from TCP Client!
```
