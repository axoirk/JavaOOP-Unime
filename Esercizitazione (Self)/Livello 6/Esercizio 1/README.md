# Esercizio 1: Java & Networking / Exercise 1: Java & Networking

## 🇮🇹 Italiano

### Descrizione
Usa la classe `URL` per connetterti a una pagina web esistente (es. Wikipedia) e usa uno `Scanner` per scaricare e stampare il codice HTML grezzo in console.

### Esempio di Output
```html
<!DOCTYPE html>
<html class="client-nojs" lang="it" dir="ltr">
<head>
<meta charset="UTF-8"/>
<title>Wikipedia, l'enciclopedia libera</title>
...
```

---

## 🇬🇧 English

### Description
Use the `URL` class to connect to an existing web page (e.g., Wikipedia) and use a `Scanner` to download and print the raw HTML code to the console.

### Sample Output
```html
<!DOCTYPE html>
<html class="client-nojs" lang="it" dir="ltr">
<head>
<meta charset="UTF-8"/>
<title>Wikipedia, the free encyclopedia</title>
...
```
