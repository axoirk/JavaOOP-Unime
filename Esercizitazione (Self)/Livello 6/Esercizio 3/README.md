# Esercizio 3: Client/Server (Datagram UDP) / Exercise 3: Client/Server (Datagram UDP)

## 🇮🇹 Italiano

### Descrizione
Riscrivi l'esercizio precedente usando i Datagram Socket per inviare pacchetti senza garantire l'ordine di arrivo (Connectionless).

### Esempio di Output
```text
[UDP Server] In ascolto...
[UDP Server] Pacchetto ricevuto: Hello UDP!
[UDP Server] Da: /127.0.0.1 porta 54321
```

---

## 🇬🇧 English

### Description
Rewrite the previous exercise using Datagram Sockets to send packets without guaranteeing the arrival order (Connectionless).

### Sample Output
```text
[UDP Server] Listening...
[UDP Server] Packet received: Hello UDP!
[UDP Server] From: /127.0.0.1 port 54321
```
