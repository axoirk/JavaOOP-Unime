# Esercizio 6 – Lista della Spesa con ArrayList / Shopping List with ArrayList

---

## 🇮🇹 Italiano

### Descrizione
Questo esercizio utilizza una struttura dati dinamica (`ArrayList`) per gestire una lista della spesa inserita dall'utente tramite tastiera.

Il programma:
- Richiede all'utente di inserire gli elementi uno alla volta.
- Aggiunge ogni elemento in un `ArrayList<String>`.
- Termina l'inserimento quando viene digitata la parola chiave `STOP`.
- Stampa infine tutti gli elementi presenti nella lista utilizzando un ciclo for-each.

### Obiettivo didattico
- Comprendere la differenza tra array statici e collezioni dinamiche come `ArrayList`.
- Gestione dell'input utente con la classe `Scanner`.
- Utilizzo di un valore sentinella per il controllo dei cicli.
- Iterazione sulle collezioni con il ciclo `for-each`.

### Esempio di Output
```text
Inserisci un oggetto nella lista della spesa: 
Latte
Oggetto aggiunto alla lista della spesa.
Per terminare l'inserimento digitare "STOP"
Pane
Oggetto aggiunto alla lista della spesa.
Per terminare l'inserimento digitare "STOP"
STOP
Lista della spesa: 
Latte
Pane
```

## 🇬🇧 English

### Description
This exercise uses a dynamic data structure (`ArrayList`) to manage a shopping list entered by the user via keyboard input.

The program:
- Asks the user to enter items one by one.
- Stores each item in an `ArrayList<String>`.
- Stops the input process when the keyword `STOP` is typed.
- Finally prints all items in the list using a for-each loop.

### Learning Objective
- Understand the difference between static arrays and dynamic collections like `ArrayList`.
- Handle user input using the `Scanner` class.
- Use a sentinel value to control loops.
- Iterate over collections with the `for-each` loop.

### Output Example
```text
Inserisci un oggetto nella lista della spesa: 
Milk
Oggetto aggiunto alla lista della spesa.
Per terminare l'inserimento digitare "STOP"
Bread
Oggetto aggiunto alla lista della spesa.
Per terminare l'inserimento digitare "STOP"
STOP
Lista della spesa: 
Milk
Bread
```
