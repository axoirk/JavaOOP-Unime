# Esercizio 2 (13 Punti) - Networking (Socket)

**Regole Appello:** 30 minuti totali per 3 esercizi. Max 5 esecuzioni per esercizio. Tutte le classi nello stesso file (Main.java).

Definire una classe `Client`. Essendo noti a priori un indirizzo IP e un numero di porta, creare un costruttore `public Client(String ip, int porta)` che, al momento dell'istanza, instauri una connessione di rete verso il server in ascolto su tali coordinate e trasmetta al destinatario un messaggio testuale arbitrario. Non sono richiesti costrutti per la ricezione della risposta.
