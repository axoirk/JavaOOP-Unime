# Esercizio 1 (5 Punti) - Logica di Base (Array)

**Regole Appello:** 30 minuti totali per l'intero esame. Max 5 esecuzioni per esercizio. Tutte le classi nello stesso file (Soluzione.java).

Implementare una classe chiamata `OperazioniArray` contenente un metodo statico con la seguente firma: `public static int contaPari(int[] array)`. Il metodo deve ricevere in input un array di interi e restituire il conteggio totale degli elementi che sono numeri pari.
