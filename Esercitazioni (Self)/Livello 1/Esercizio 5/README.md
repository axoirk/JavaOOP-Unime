# Esercizio 5

---

### Descrizione
Crea due classi in due file separati. Inserisci la prima in un package specifico usando la direttiva `package` all'inizio del file, e importala nella seconda classe usando `import`.

#### Requisiti:
- Definire una classe di utilità in un package (es. `com.math.Utils`).
- Creare una classe `Main` in un altro package che utilizzi la classe di utilità.

### Esempio di Output
```
Risultato dell'operazione inter-package: 25
```
