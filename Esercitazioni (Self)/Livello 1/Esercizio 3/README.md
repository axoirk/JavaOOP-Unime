# Esercizio 3

---

### Descrizione
Usa un costrutto `switch` basato su costanti o enumerazioni per stampare un messaggio diverso in base al voto (A, B, C, D, F) di uno studente.

#### Requisiti:
- 'A': "Eccellente"
- 'B': "Molto Buono"
- 'C': "Sufficiente"
- 'D': "Scarso"
- 'F': "Bocciato"
- Gestire il caso `default` per voti non validi.

### Esempio di Output
```
Inserisci il voto dello studente (A-F): B
Valutazione: Molto Buono
```
