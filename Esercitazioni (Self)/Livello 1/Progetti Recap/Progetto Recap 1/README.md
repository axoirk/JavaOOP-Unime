# Esercizio 1

**Difficoltà:** Facile 🟢
**Tempo Max:** 10 minuti
**Compilazioni Max:** 5

**Traccia:**
Dato un anno intero in input, determinare se è bisestile. Un anno è bisestile se è divisibile per 4 ma non per 100, oppure se è divisibile per 400.

**Dati in Input (Hardcoded o tramite Scanner):**
`int anno = 2024;`

**Output Atteso:**
`L'anno 2024 è bisestile.`

*(Test opzionale: se l'input fosse `1900`, l'output dovrebbe essere `L'anno 1900 NON è bisestile.`)*
