# Esercizio 3

**Difficoltà:** Difficile 🔴
**Tempo Max:** 10 minuti
**Compilazioni Max:** 5

**Traccia:**
Creare un programma all'interno del package `org.esame.analisi`. Il programma deve analizzare una sequenza di caratteri per contare e restituire il numero totale di vocali minuscole (a, e, i, o, u). Il calcolo si interrompe non appena viene intercettato il carattere sentinella `#`. Qualsiasi carattere o vocale posizionato dopo il `#` deve essere scartato.

**Dati in Input:**
`char[] input = {'p', 'r', 'o', 'g', 'r', 'a', 'm', 'm', 'a', 'z', 'i', 'o', 'n', 'e', '#', 'a', 'b', 'c'};`

**Output Atteso:**
`Totale vocali: 6`
