# Esercizio 1

**Difficoltà:** Facile 🟢
**Tempo Max:** 10 minuti
**Compilazioni Max:** 5

**Traccia:**
Trasferire una sequenza di messaggi testuali verso una destinazione fisica sul file system.

**Dati in Input:**
Vettore di stringhe: `{"Avvio", "Errore", "Chiusura"}`
Nome destinazione: `log.txt`

**Output Atteso:**
Creazione del file `log.txt` contenente i messaggi riga per riga. A console: `Log scritto con successo.`
