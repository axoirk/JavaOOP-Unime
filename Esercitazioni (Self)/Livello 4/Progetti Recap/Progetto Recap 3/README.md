# Esercizio 3

**Difficoltà:** Difficile 🔴
**Tempo Max:** 10 minuti
**Compilazioni Max:** 5

**Traccia:**
Progettare un'interfaccia grafica basilare. Deve presentare un'area per l'inserimento libero di testo e un bottone. L'attivazione del bottone deve provocare il dump fisico del testo digitato all'interno di un file di sistema, mascherando eventuali difetti I/O a console.

**Dati in Input:**
Digitazione nell'area di testo della stringa: "Appunti esame" e successiva pressione del tasto.

**Output Atteso:**
Generazione del file di testo fisico contenente "Appunti esame". Nessun output obbligatorio a console.
