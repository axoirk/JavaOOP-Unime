# Esercizio 2

**Difficoltà:** Media 🟡
**Tempo Max:** 10 minuti
**Compilazioni Max:** 5

**Traccia:**
Salvare su file binario lo stato esatto di un Personaggio giocabile (nome e punteggio). È severamente vietato includere nel salvataggio binario i dati sensibili (password). Recuperare l'oggetto dal file binario e testarne il contenuto.

**Dati in Input:**
Oggetto Personaggio: "Zelda", punteggio: 100, password: "secret_pass"

**Output Atteso:**
`Oggetto ricaricato: Zelda, Punteggio: 100, Password: null`
