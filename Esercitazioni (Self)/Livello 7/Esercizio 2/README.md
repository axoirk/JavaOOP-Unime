# Esercizio 2: Gestione Dipendenze e JDBC

### Traccia
Inserisci nel `pom.xml` le dipendenze per connetterti a un database relazionale usando JDBC (es. driver MySQL o HSQLDB) e testa la connessione.

### Esempio di Output
```text
Connecting to database...
Connection successful!
Executing test query: SELECT 1
Result: 1
Closing connection.
```
