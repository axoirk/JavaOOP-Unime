# Esercizio 3

**Difficoltà:** Difficile 🔴
**Tempo Max:** 10 minuti
**Compilazioni Max:** 5

**Traccia:**
Impalcare un webserver autonomo capace di erogare risorse REST in formato JSON all'interrogazione di uno specifico URI.

**Dati in Input:**
Richiesta client su `GET /api/studenti`

**Output Atteso:**
Restituzione di un array JSON simile a: `[{"nome":"Mario","matricola":123}, {"nome":"Luigi","matricola":124}]`
