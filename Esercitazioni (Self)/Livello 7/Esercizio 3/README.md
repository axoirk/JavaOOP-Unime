# Esercizio 3: Integrazione NoSQL (MongoDB)

### Traccia
Sostituisci il database relazionale aggiungendo al POM la dipendenza `mongo-java-driver`. Scrivi un file nel DB documentale.

### Esempio di Output
```json
{
  "_id": { "$oid": "60d5ecb5f2a1b1a2c3d4e5f6" },
  "name": "Student A",
  "course": "OOP",
  "level": 7
}
```
*Documento salvato e riletto correttamente dal database MongoDB.*
