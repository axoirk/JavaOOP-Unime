# Esercizio 1 – Ereditarietà

---

### Descrizione
Questo esercizio esplora i concetti fondamentali dell'**ereditarietà** e della relazione "is-a" tra classi.

Il programma:
- Definisce una superclasse `Person` con attributi base come il nome.
- Estende `Person` con una sottoclasse `Student`, aggiungendo attributi specifici come il numero di matricola.
- Dimostra l'uso di `super()` per richiamare il costruttore della superclasse.
- Illustra il **polimorfismo**: un riferimento di tipo `Person` può puntare a un oggetto di tipo `Student`.

### Obiettivo didattico
- Comprendere la gerarchia delle classi in Java.
- Utilizzo della keyword `extends`.
- Override dei metodi per specializzare il comportamento delle sottoclassi.

### Esempio di Output
```text
Name: Giulia
Name: Marco
Student Number: 12345
```
