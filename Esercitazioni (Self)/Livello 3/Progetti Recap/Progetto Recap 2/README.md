# Esercizio 2

**Difficoltà:** Media 🟡
**Tempo Max:** 10 minuti
**Compilazioni Max:** 5

**Traccia:**
Stabilire un contratto per l'esecuzione di Pagamenti. Implementare il contratto per emulare una transazione. Qualora l'importo della transazione superi la soglia critica di 1000, l'esecuzione deve essere interrotta sollevando un'eccezione customizzata (che obblighi il compilatore al controllo) denominata `SaldoInsufficienteException`.

**Dati in Input:**
Tentativo di transazione con importo 1200.

**Output Atteso:**
`Errore di Transazione: Saldo Insufficiente per elaborare 1200.`
