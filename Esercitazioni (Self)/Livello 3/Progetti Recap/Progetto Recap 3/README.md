# Esercizio 3

**Difficoltà:** Difficile 🔴
**Tempo Max:** 10 minuti
**Compilazioni Max:** 5

**Traccia:**
Definire uno scheletro base per un Dispositivo. Creare l'implementazione per un Computer. In fase di avvio, se il parametro RAM è inferiore a 8, sollevare un errore a runtime irreversibile. Gestire completamente la casistica di avvio (con RAM insufficiente), garantendo sempre e comunque il rilascio incondizionato delle risorse hardware, stampandolo a schermo.

**Dati in Input:**
Avvio computer con RAM = 4.

**Output Atteso:**
`Tentativo di avvio...`
`Errore critico: RAM insufficiente.`
`Risorse hardware rilasciate.`
