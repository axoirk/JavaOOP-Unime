# Esercizio 5: Java e XML (SAX)

### Descrizione
Usa un parser a eventi (`SAXParserFactory` e `DefaultHandler`) per leggere lo stesso file XML senza caricarlo tutto in memoria, catturando i metodi `startElement` ed `endElement`.

### Esempio di Output
```text
Informazioni Persona: Mario Rossi (1234)
Informazioni Persona: Elena Bianchi (5678)
Informazioni Persona: Giuseppe Verdi (9012)
Informazioni Persona: Anna Rossi (3456)
```
