# Esercizio 1

**Difficoltà:** Facile 🟢
**Tempo Max:** 10 minuti
**Compilazioni Max:** 5

**Traccia:**
Apertura di un canale HTTP puro verso un indirizzo internet noto. Ricezione e stampa del payload crudo per i primi segmenti di risposta.

**Dati in Input:**
Endpoint: `http://example.com`

**Output Atteso:**
Stampa a console delle primissime 5 righe (o segmenti) del codice sorgente HTML ricevuto dalla pagina.
