# Esercizio 3

**Difficoltà:** Difficile 🔴
**Tempo Max:** 10 minuti
**Compilazioni Max:** 5

**Traccia:**
Eseguire il parsing di un file formattato strutturalmente e caricare i nodi in memoria (Document Object Model). Ricercare e stampare attributi specifici relativi a credenziali di backend.

**Dati in Input:**
File testuale `config.xml` situato a runtime nella directory, con struttura:
`<config><url>localhost</url><user>root</user><password>1234</password></config>`

**Output Atteso:**
`URL: localhost`
`User: root`
`Password: 1234`
